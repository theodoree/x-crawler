**新工具/技术**（如有）
- baoyu-cover-image：dotey 提供的“生图 skills”之一，用于自动生成文章封面，支持自定义风格与输出比例（示例建议 2.35:1）。
- baoyu-article-illustrator：用于根据文章内容分段生成插图，自动把段落映射为图像场景（示例建议 16:9）。
- Claude + 插件 工作流：通过 Claude Code 调用插件并“@你的文章”，让模型根据文章语义生成图片提示词和素材分段。
- lovart / nano banana pro：落地的图像生成平台，作者提到可用每日免费积分（示例每天 80 积分），适合低成本批量生成。

**核心观点/方法论**（如有）
- 把“文章 → 分段提示 → 图片”当作自动化生产线，AI 能根据语义拆分并为每段生成合适插图，显著提升内容生产效率和一致性。
- 明确风格与长宽比（用通用风格提示词）能保证视觉统一，便于批量和品牌化输出。
- 先生成并保存提示词而非直接调用付费API，可节省成本并便于复用与迭代优化。
- 将喜欢的风格让AI抽象成一段通用提示词，下次直接套用可快速复刻同一视觉体系。

**实践经验/案例**（如有）
- 实操流程：安装 dotey 的生图 skills → 在 Claude code 中用插件 @文章 触发 baoyu-article-illustrator → 审核/微调生成的提示词与输出。
- 成本控制技巧：作者未接生图 API，先把生成的提示词保存到文件，再用 lovart 等免费积分平台批量生成；多个账号可叠加额度。
- 封面与插图建议：封面 2.35:1，插图 16:9；可在每幅图内加入一句“金句”或关键元素以增强信息传达。
- 风格复用策略：对满意的一套图让AI总结成通用风格提示，之后直接用该提示生成同风格图集，节省重复设计成本。
- 小结：dotey 的分享覆盖从技能包、提示工程到低成本生成落地的完整流程，适合内容创作者做规模化、风格一致的图像生产。
