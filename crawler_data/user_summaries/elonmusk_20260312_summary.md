## 核心动态
- Grok 4.20 Beta 发布：引入 2,000,000 令牌上下文窗口、低幻觉率与显著加速，并提供 Multi‑Agent / Reasoning / Non‑Reasoning 三种 API 变体，表明 xAI 在大上下文推理与多代理场景上取得行业竞争优势。  
- Grok Imagine 大更新：现在可以在生成视频时引用视频并整合多张参考图（报道提到 5–7 张），显著提升动作、风格与角色一致性的影像生成能力。  
- 强烈的市场与社区认可：Grok 在 App Store/Google Play 获得海量高评分评论（App Store 过百万评论，Play 超 200 万且 4.8 星），大量用户作品和正面反馈推动口碑扩散。  
- Digital Optimus 与特斯拉协同推进：宣称 AI4 装备车辆可在闲置时“做办公任务”，并计划在超充站部署大量 Digital Optimus 单元，显示 xAI 与特斯拉在感知（Tesla）+ 推理（xAI）整合上的落地路线。

## 值得关注
- 2,000,000 token 是明确卖点：相比 GPT‑5.4 的 1M 与 Claude 的 200K，这对长篇文档、复杂对话或多步代理尤为重要。  
- 性能指标亮眼：公开数据称 Grok 4.20 在 AA‑Omniscience 幻觉率约 22%，IFBench 指令遵从 ~82.9%，输出速度约 265 tokens/s。  
- 参考图数量不同口径：有帖称可用到 7 张参考图，也有说法写 5 张，用户需注意不同界面/版本的具体限制。  
- 内容审核口径偏宽：Elon 表示“R‑级电影允许的内容在 @Grok Imagine 也允许”，可能带来创作自由与合规/安全的张力。  
- 社区示例丰富：多条转发展示多镜头一致性、视频扩展、幽默输出等，创作者已经在实测并分享高质量成果。  
- 快速迭代节奏：Grok 4.20 Heavy（Beta 2）侧重速度与深度分析，官方称 Beta 3 将带来修复与功能增强，体现频繁上线的策略。  
- 产品声量与政治议题并行：除了产品推广，账号大量转发关于选举法（Voter ID / SAVE Act）与 OpenAI 安全争议（学校枪案相关）的内容，政治话题占比高。  
- 航天里程碑同步曝光：转发提到 DART 成功改变小行星轨道，以及即将使 Starlink 卫星数突破 10,000 的双重发射计划。  
- 教育与安全并存：有推文鼓励教孩子用 Grok Imagine（含儿童模式），但同时也出现“把 ChatGPT 远离儿童和心理不稳者”的警告，反映推广与风险提醒并存。  
- 当天信息以放大社区内容为主：大量 RT 与用户作品被转发，原创长文较少，倾向于宣发产品更新与展示用户案例。
